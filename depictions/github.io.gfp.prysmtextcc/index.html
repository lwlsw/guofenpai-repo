Prysm TextCC
Simple text style for your control Center.
This also works on normal CC too
I Added 5 Presents for MagmaEVO (Automatically install in Preset directory while installation)
I made pretty simple things here for easy to use.
Installation process - it just take 5 minutes
1) After installing TextCC go in Folder location using filza Library/Themes/PrysmLargeTextCC.theme or PrysmSmallTextCC.theme/bundle you'll find prysm folder copy that folder (choose between Large or Small version which you like)
2) paste that folder in Library/Application Support folder.
3) apply TextCC using snowboard or ithemer then respring device. (If you using snowboard then must install snowboard ui extension link for repo in important notes section)
4) Adjust toggles from control center settings which is with black background icons (must install Flipconvert tweak link for repo in important notes section)
5)Now respring your device.
Here Video tutorial if you not understand in those steps
That's it it's so simple.
