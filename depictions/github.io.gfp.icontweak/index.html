Main Features

By swiping the icon on the SpringBoard (or tap in Editing mode / Force Touch menu), you can:

  • Open in Filza [App]
  • Open in Filza [Data]
  • Rename App
  • Clear Badges
  • Set Custom Badges
  • Clear Caches
  • Backup Data
  • Restore Data
  • Delete Backup Data
  • Remove All User Data
  • Copy BundleID
  • Copy URL Schemes
  • Mute Notification
  • Open App

The custom badge must be enabled in the Settings notification.

Option

  • Show App Cache Size
  • Use Text Badges
  • Hide Icon Labels

Clear Caches displays the size of the cache files.
Text badges do not display many strings.
