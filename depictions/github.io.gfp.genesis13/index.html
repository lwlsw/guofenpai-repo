Genesis is supported on all devices running iOS 13.0 to 13.5! If you are having any issues with a certain feature not working on your device, please feel free to contact me on Twitter!

Genesis

Genesis is a tweak that aims to allow users to customize the things that Apple doesn't normally allow. It's made for people who are tired of waiting for Apple to give us a redesign.

Features

The latest version of Genesis includes the following customization options:
1. Battery Options
2. Control Center Options
3. Lock Screen Options
4. Power Menu Options
5. SpringBoard Options (Includes Status Bar)
6. Other (System-Wide) Options

  • Added Feature: Hide Tab Bar Labels
