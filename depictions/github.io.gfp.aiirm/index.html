AiirM Theme Bundle

AiirM - A Minimalistic Style theme bundle!
Custom made StatusBar, Page Dots, Notification Badges, Docks, Messages, LockScreen and Much More!
All in one theme for your device! Everything is categorized in Snowboard for ease of use. Enable only what you want, icons no longer match your new setup? Change it up whenever you want!
All icons made by me, with multiple other designs in the works!

Caution:

Tested only on iPhone X, If you have any issues with any icons please contact me. Especially Non-Notched devices as I wasn't able to test. Signal Icons for DualSim requires NiceBarX. Sorry for the inconvenience.

Requirements:

Snowboard:

  • Must have Snowboard and Snowboard Extensions.
  • It's recommended to install all the extensions to avoid issues in the future.

iThemer: (alternative for snowboard)
NiceBarX (alternative to theme statusbar elements)
Juice (Battery themes)
BubbleLineRemover (available on Twickd). This is required for some message bubble themes to appear properly.

  • NOTE: BubbleLineRemover OR DopeConvos (You dont need both) Will solve this issue with the background in the bubbles. BUT Dopeconvos is recommended as the transparent bubbles will only work properly in darkmode. Light mode the text will be white / unreadable. Dopeconvos can solve this by changing the text colour.
  • Using Snowboard UI, Long press and disable message bubble.

What you get:

  • Multiple different Signal icons to pick from
  • Multiple different Wifi icons to pick from
  • Multiple different Battery icons to pick from
  • Multiple different Page Dots icons to pick from
  • Multiple different Notification Badges to pick from
  • LockScreen Flashlight and Camera Icons
  • Multiple Docks
  • Multiple Messages Bubble Themes (Requires Snowboard UI Extension and BubbleLineRemover / DopeConvos)
  • Too many icons to show in screenshots!

Mix and match any icons together without any restrictions!

NOTE: This theme DOES NOT include any APP Icons. App Icon theme used in screenshots is "Viola", Mask and AE from AiirX.

Battery Theme Setup Instructions:

To get the battery theme to apply you need have Juice.
Juice:

  • Click Select Style and pick the style you want.
  • Click Edit > Settings > Adjust icon for your device if needed

Done!
