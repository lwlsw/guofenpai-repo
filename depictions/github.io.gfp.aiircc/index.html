Aiir CC Theme

Custom made Control Center Theme. Comes with 2 different sizes to best match your current theme and preference.
Default icons are white. You will need a tweak like "Magma Evo" to recolour them.
Two Styles Available:

  • Glyph Icons (Big/Small)
  • Circle Icons (Big/Small)

If you have any issues with icons, please contact me and I will try to resolve as fast and best as possible.

Requirements

Must have Snowboard and Snowboard UI Extensions.
(Found on Sparks Discord https://discord.gg/aYgD69T ) Latest version is recommended
In his discord > Snowboard-test-builds > you should see a pinned DEB for snowboard UI (ex: com.spark.snowboard.ui_1.0.41Beta2_iphoneos-arm.deb)
save it with Filza and install it.
Note: If you see this version shown in the example above in his repo, install it from there.

Instructions - Read Carefully


You must have:

  • CCModules
  • FlipConvert - REPO: https://julioverne.github.io/https://julioverne.github.io/
  • FlipSwitches - REPO: http://rpetri.ch/repo/http://rpetri.ch/repo/
  • SnowBoard
  • SnowBoard UI Extension (Found on Sparks Discord https://discord.gg/aYgD69T )
  • iThemer (Use either iThemer OR Snowboard + Snowboard UI Extension)

Theming CC Toggles:

  • Open Snowboard > AiirCC > (Style & Size)
  • OR
  • Open iThemer > Enable > AiirCC > (Style & Size)

You do not need both iThemer and Snowboard
Note: Default stock Control Center will not theme your Wifi and Bluetooth icon for that you will need "BetterCCiconsPro" this is the only workaround i know. You do not need this with Prysm as it solves that issue.
Theming CC Animated Toggles:

  • Navigate to Settings App>Control Centre>Customize Controls.
  • Add the new FlipConvert toggle which has the theme
  • Respring

Magma Evo Theme:

  • Copy all text > Magma Evo > Import
