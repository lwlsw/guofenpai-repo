Prysm CalmCC
Unique customization to Your Control Centre.
This also works on normal Control center too
I Added 5 Presents for MagmaEVO (Automatically install in Preset directory while installation)
I made pretty simple things here for easy to use.
Installation process - it just take 5 minutes
1) After installing CalmCC go in Folder location using filza Library/Themes/PrysmCalmCC.theme/bundle you'll find prysm folder copy that folder
2) paste that folder in Library/Application Support folder.
3) apply CalmCC using snowboard or ithemer then respring device.
4) Adjust toggles from control center settings which is with black background icons
5)Now respring your device.
Here Video tutorial if you not understand in those steps
That's it it's so simple.
